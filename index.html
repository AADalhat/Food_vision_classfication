🍲 FoodVision

AI-Powered Nigerian Food Classifier

🔍 What It Does

FoodVision is a deep learning-powered web app that classifies traditional Nigerian dishes and provides their ingredients, nutrition, ethnicity, and health tips.

🌐 Try it live on Hugging Face

🧠 Tech Stack

  • Gradio for UI
  • JSON metadata system
